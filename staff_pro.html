Nigeria Staff Promotion Algorithm

Project Information

  • Category : Machine Learning
  • Project Date : Jul, 2020
  • Project URL : https://github.com/Utkarsh217/Nigeria-Staff-Promotion-Algorithm

What's the Project about!!!

Built a machine learning model to predict staff , that are likely to be promoted based on defined personal and performance parameters like number of companies that an employee worked with before joining the organisation , measure of employees who meet the annual set target , previous year overall performance HR score , an indicator if a staff has been summoned to a disciplinary panel in the past and some more performance parameters.

The model is evaluated by the harmonic mean of precision and recall values used for a classification problem where imbalanced dataset is observed.
